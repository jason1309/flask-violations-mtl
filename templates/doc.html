Retourne les contrevenants entre les dates passées en paramètre.

  • Response

Query Parameters

post /api/plainte

Créer une nouvelle plainte sur un établissement.

  • Request
  • Response

Body

Media type: application/json

Type: object

Properties
  • nom_etablissement: required(string)
  • adresse: required(string)
  • ville: required(string)
  • date_visite: required(string)
  • nom_client: required(string)
  • prenom_client: required(string)
  • description: required(string)

Example:

{
  "nom_etablissement": "Subway",
  "adresse": "1000 av. Chambéry",
  "ville": "Mascouche",
  "date_visite": "2023-04-10",
  "nom": "Gagné",
  "prenom": "Jason",
  "description": "J'ai vu 2 rats."
}

HTTP status code 201

La plainte s'est créée avec succès.

Body

Media type: application/json

Type: object

Properties
  • id: required(integer)
  • nom_etablissement: required(string)
  • adresse: required(string)
  • ville: required(string)
  • date_visite: required(string)
  • nom_client: required(string)
  • prenom_client: required(string)
  • description: required(string)

Example:

{
  "id": 1,
  "nom_etablissement": "Subway",

  "ville": "Mascouche",
  "date_visite": "2023-04-10",
  "nom": "Gagné",
